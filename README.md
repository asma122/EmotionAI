# 🧠 ScanEmotion – Multimodal Emotion Detection System
This project is part of a research-oriented internship combining **AI, healthcare, and human emotion understanding**.

---

## ⚙️ Technologies

The system integrates a **multimodal AI stack** combining deep learning, NLP, computer vision, and full-stack development:

### 🤖 Artificial Intelligence & Deep Learning
TensorFlow, Keras, Hugging Face Transformers

### 🧠 Natural Language Processing (NLP)
TextBlob, NLP preprocessing pipelines

### 🎥 Audio & Video Processing
OpenCV, MediaPipe, Librosa, SpeechRecognition

### 📊 Data Handling & Scientific Libraries
NumPy, Pickle

### 🌐 Backend & Database
Flask, SQLAlchemy, SQLite

### 📄 Reporting System
PDFKit (automated PDF report generation)

### 💻 Programming Language
Python
